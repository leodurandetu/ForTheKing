Detailed Description

Gestion centralisée des ressources graphiques et textuelles.

Author
Massoud

Fournit la structure et les fonctions nécessaires pour charger, stocker et libérer toutes les textures et polices utilisées dans le jeu.

int charger_ressources ( SDL_Renderer * renderer,
ressources_jeu_t * res,
perso_type_t type_perso )

Charge l'ensemble des ressources du jeu en mémoire.

Lit les fichiers d'assets (images, polices) sur le disque et génère les textures SDL correspondantes.

  • Parameters
    rendererLe moteur de rendu SDL utilisé pour créer les textures.
    resPointeur vers la structure ressources_jeu_t à peupler.
    type_persoLe type du personnage à charger.
  • Returns
    0 en cas de succès, un code d'erreur (-1) si un fichier est manquant.

void liberer_ressources ( ressources_jeu_t * res)

Libère la mémoire allouée pour les ressources du jeu.

Détruit toutes les textures et ferme les polices contenues dans la structure pour éviter les fuites de mémoire. À appeler avant de quitter.
